# 스티커 메모

# 기능
- 메모지 디스플레이에 메모를 작성
- 메모지는 drag & drop으로 이동이 가능
- 메모지에 작성된 내용은 자동으로 저장되어 새로고침을 해도 내용유지
- 추가 버튼을 만들어서 메모지 추가 기능
- 메모지 디스플레이 크기 조절

# 사용 기술
- react
- react-dom
- typescript
- mobx, mobx-react
- mui/icons-material, mui/material
- emotion/react, emotion/styled
